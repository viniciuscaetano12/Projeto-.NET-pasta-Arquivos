# Projeto-.NET-pasta-Arquivos


Criando o projeto Arquivos
 Abra uma pasta de preferência em seu computador; na 
sequência, abra o terminal de comando na pasta 
exemplificada. Lembre-se de que você pode navegar entre as 
pastas de seu computador com os comandos “cd ..”, para voltar 
à pasta anterior, e “cd nome_pasta”, para entrar na pasta correspondente 


Para criar o projeto, utilizaremos o comando aplicado 
anteriormente, dotnet new console -o Arquivos.
